## Overview

This project is a responsive awareness and resource website about Türkiye's ongoing earthquake challenges. It helps visitors understand the risk through a live interactive fault-line/earthquake map, learn practical preparation and safe-response steps, and see how earthquakes impact infrastructure. The site also shares resilience stories and trusted resources.
