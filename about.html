About This Project

This website was created for an INFS 2400 international project to raise awareness about Turkey's ongoing earthquake challenges. The goal is to make the science and reality of earthquakes easier to understand, especially why Turkey is at such high risk and how those risks affect people, buildings, and communities across the country.

The site combines real-time earthquake data with an interactive map centered on Turkey, along with clear visual charts and a detailed table of recent events. Visitors can explore where earthquakes are happening, how strong they are, and which regions are most affected. Other sections focus on building safety, infrastructure impacts, and practical preparation tips so people can better understand what to do before, during, and after an earthquake.

This project was built using HTML, CSS, JavaScript, and Bootstrap. It uses live earthquake data from the USGS (United States Geological Survey) API and turns raw numbers into an interface that is more visual, interactive, and easier to learn from. Overall, the site is meant to be both an educational tool and a reminder that earthquake awareness and preparation are ongoing needs for Turkey and for anyone living in seismic regions.
